# Laravel Historiable

Easily keep history of your Laravel models.

## Version Compatibility

 Laravel      | laravel-historiable
:-------------|:-------------------
 5.5.x|5.6.x  | 0.2.x
 5.4.x        | 0.1.x


## Installation

```
$ composer require luisdalmolin/laravel-historiable:"0.2.*"
```
