# Laravel Historiable

Easily keep history of your Laravel models.

## Version Compatibility

 Laravel  | laravel-historiable
:---------|:----------
 5.4.x    | 0.1.x


## Installation

```
$ composer require luisdalmolin/laravel-historiable:"0.1.*"
```
